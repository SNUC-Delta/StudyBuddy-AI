# StudyBuddy-AI

## Starting the Client
Yarn: ```yarn dev```
NPM: ```npm dev```

## Starting the Server
Command: ```python ./server/server.py```

## Installing the Requirements
1. Setup Pytorch Locally
2. Run Command: ```pip install -r ./server/requirments.txt```
3. Client Requirements -- ```yarn install``` or ```npm i```
